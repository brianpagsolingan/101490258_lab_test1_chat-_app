***Lab test 1 chat app***
When the app is ran, you will be directed to a login page and will be prompted to input a username and password. There is a button that redirects you to sign up page.
Where you will be prompted to input a unique username, password, name, and lastname. Once logged in you will be able to choose a chat room. You can then use the chatroom where your messages will be saved in the database.
You can either log out from here or change chat rooms.
